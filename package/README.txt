医美诊所管理系统 v1.3
======================

安装方法（三选一）
--------------------

方法 A：双击安装（最快）
  1. 双击「install.command」
  2. 如出现安全提示，请右键 → 打开 → 点击「打开」
  3. 安装程序会自动安装 PostgreSQL（首次需联网）
  4. 完成后从「应用程序」启动「Clinic.app」

方法 B：Terminal 安装（推荐，避开安全提示）
  1. 打开 Terminal（终端）
  2. 输入以下命令（把路径拖入 Terminal 可以自动填入）：
        cd /path/to/clinic-mgmt-dist
        chmod +x install.sh && ./install.sh

方法 C：手动安装
  1. 把 Clinic.app 拖到 /Applications
  2. 打开 Terminal 运行：
        xattr -rd com.apple.quarantine /Applications/Clinic.app
  3. 从「应用程序」启动

局域网部署
----------
在一台电脑上安装运行后，局域网内其他电脑可通过浏览器访问：

1. 在服务器电脑上双击启动 Clinic.app
2. 终端窗口会显示局域网地址，例如：http://192.168.1.100:8080
3. 其他 4 台电脑的浏览器输入该地址即可使用
4. 所有数据保存在服务器电脑本地，无需额外配置

注意事项
--------
- 服务器电脑建议设置固定 IP（路由器后台设置）
- 如无法访问，检查服务器电脑的防火墙是否放行 8080 端口
  - 系统设置 → 网络 → 防火墙 → 选项 → 添加 Clinic.app
- 首次使用需在服务器电脑上完成安装流程（自动安装 PostgreSQL）
- 客户端电脑无需安装任何软件，仅需浏览器（推荐 Chrome）
